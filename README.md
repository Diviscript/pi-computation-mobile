# π Computation Mobile

High-precision computation of π (pi) using Python and mpmath, optimized to run on Android devices via Termux.

## Features
- High-precision π calculation
- Checkpoint system (resume computation)
- Performance monitoring (digits/sec, ETA)
- SHA256 verification
- BBP hexadecimal digits generation

## Requirements
- Python 3
- mpmath
- Termux (Android)

## Usage
```bash
python pi_master.py
